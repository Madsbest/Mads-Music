🎵 MadsMusic
MadsMusic is een moderne, minimalistische muziekspeler voor Android. Gebouwd met Python en Kivy, biedt het een lichtgewicht alternatief voor zware streamingdiensten, met de focus op lokale muziekopslag en een strakke gebruikerservaring.

✨ Kenmerken
Minimalistisch Design:
- Een unieke interface met het kenmerkende groene "M-noot" icoon.
- Lichtgewicht: Geoptimaliseerd om weinig RAM en batterij te verbruiken.
- Android Service: Muziek blijft doorspelen, ook als de app op de achtergrond draait of het scherm uitstaat.
- Geen Advertenties: Een pure luisterervaring zonder onderbrekingen.

🛠️ Technische Stack
- Taal: Python 3.11
- GUI Framework: Kivy
- Android Bridge: Pyjnius (voor interactie met Android API's)
- Build Tool: Buildozer
- Audio Engine: Kivy SoundLoader (geoptimaliseerd voor mobiel)
